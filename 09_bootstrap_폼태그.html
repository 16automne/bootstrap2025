부트스트랩 02_폼태그

다양한 폼 사용을 위한 폼 컨트롤 스타일, 레이아웃 옵션, 사용자 정의 컴포넌트의 예시와 사용 가이드입니다.

폼태그 사용시 form, fieldset, legend태그 사용.

폼양식 제목 (로그인, 회원가입 ...)

(비밀번호는 특수문자를 포함 8-10까지 가능)

class='form-text'는 양식 작성시 설명문구를 넣고자할 경우 사용

폼입력양식의 크기조정은 .form-control-lgm ,form-control-sm 으로 크기 조정 가능

.form-text 사용시 block요소와 간격을 쉽게 맞출 수 있도록 상단 여백이 추가. small, span를 사용하면 좋음

버튼 비활성화 - disable

disabled 불리언 속성 추가시 비활성화 외형이되며, 포인트 이벤트가 삭제되며 포커싱X

읽기전용 - readonly

readonly불리언 속성을 추가함으로써 입력값의 변경 막기 가능. disabled와 달리 readonly 입력값은 계속 포커싱O 선택O

라디오 버튼 - lable태그에 .form-check-label, input태그에 .form-check-input을 적용

체크박스 - lable태그에 .form-check-label, input태그에 .form-check-input을 적용 (라디오버튼과 동일)

취미선택:

선택목록 - select, option

class="form-select"적용시 width: 100%;로 커짐. .form-select-lg or .form-select-sm 서식 적용 가능

selectㅐ그에 multiple속성 삽입시 다중 선택도 가능. size속성으로 크기 조정가능.

다중 텍스트 박스 - 높이 서식 css로 적용 가능

파일첨부

.input-group안에서 input, label태그의 순서를 변경하면 'upload'제목의 앞/뒤 순서가 변경됨
