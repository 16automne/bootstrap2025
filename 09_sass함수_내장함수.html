@function 문법

  • 사용자가 함수를 직접 정의해 사용가능 문법
  • @mixin과 비슷해보이지만 반환되는 내용이 다름
  • @mixindms 지정한 스타일을 반환하지만, @function(함수)는 연산된 값을 @return키워드로 반환
    문법)
    @mixin 믹스인명($매개변수){
      스타일 서식
    }
    ----------------------------
    @function 함수명($매개변수){
      @return 값;
    }
    ----------------------------
    "호출 방법"
    @mixin기법은 호출시 '@include 믹스인이름(인수);' 처럼 사용하고, 함수문법은 호출시 '함수명(인수)'으로 호출해 사용.
