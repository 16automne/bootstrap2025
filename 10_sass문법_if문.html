2. @if문법

자바스크립트의 if문과 비슷하게 사용

@else를 붙여 false일 경우 실행하고 다중if문도 실행 가능

    문법)
    @if(조건){
      //참 표현식
    }
    ---------------------
    @if(조건식){
      //참 표현식
    }@else{
      //거짓 표현식
    }
    ---------------------
    @if(조건식1){
      //조건식1이 참일때 표현식
    }@else if(조건식2){
      //조건식2이 참일때 표현식
    }@else{
      //거짓 표현식
    }
