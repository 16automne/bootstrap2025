1. @for문

@for 키워드는 '스타일을 반복적'으로 출력하는데 사용

javascript의 for문과 유사

    문법1)
    through: 종료 만큼 반복
    @for $변수 from 시작 through 종료{
      //반복할 내용
    }

    문법2)
    to: 종료 전까지 반복
    @for $변수 from 시작 to 종료{
      //반복할 내용
    }

$변수 => #{변수}

#{변수}는 코드 어디든지 변수 값을 사용할 수 있음

h1{background-image:url('./img/#{$img}');}
