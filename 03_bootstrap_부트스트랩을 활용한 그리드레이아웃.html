03_부트스트랩을 활용한 그리드레이아웃

부트스트랩은 하나의 행에 반드시 12개의 열(컬럼)을 사용해야 합니다.

만약 12개의 열(컬럼)을 모두 사용하지 않으려면, 더 넓은 열(컬럼)을 만들기 위해 열(컬럼)들을 하나의 그룹으로 만들어 사용할 수 있습니다.

그리드 시스템에서는 화면의 너비별 5개의 클래스를 가지고 있습니다. 이미지 참고

해상도별 클래스명

  • 해상도 < 576 : class(X)
  • 해상도 >= 576 : class="sm" (작은모바일.ver)
  • 해상도 >= 768 " class="md" (작은테블릿.ver)
  • 해상도 >= 992 " class="lg" (큰테블릿.ver)
  • 해상도 >= 1200 " class="xl" (일반PC.ver)
  • 해상도 >= 1400 " class="xxl" (큰PC.ver)

그리드 레이아웃 실습

12칼럼을 한줄에 배치

lg: 가로해상도 992px이상일 경우 가로 영역을 12칼럼으로 인식(설정)













12칼럼을 4칸씩 병합후 배치

md: 가로해상도 768px이상일 경우 가로 영역의 칼럼을 3등분(4칸을 1칸으로 병합)해 인식













12칼럼을 6칸씩 병합후 배치

sm: 가로해상도 576px이상일 경우 가로 영역의 칼럼을 2등분(6칸을 1칸으로 병합)해 인식
